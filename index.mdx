Nex provides a powerful REST API for sharing real-time organizational context with AI agents. Create and manage custom objects, records, relationships, tasks, notes, and more.

## Quick Start

```bash
# Authenticate with your API key
curl https://app.nex.ai/api/developers/v1/objects \
  -H "Authorization: Bearer sk-YOUR_API_KEY"
```
