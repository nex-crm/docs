## Welcome

The NEX CRM Developer API provides programmatic access to your CRM data. Generate API keys from the NEX web UI and use them to authenticate requests.

## Authentication

All API endpoints are authenticated using API keys. Include your API key in the request headers:

```http
X-API-Key: your_api_key_here
```

The API uses the following security scheme as defined in the OpenAPI specification:

```json
"security": [
  {
    "ApiKeyAuth": []
  }
]
```

## Base URL

The API is available at: `https://api.nex.ai/api`

## Available Endpoints

The API provides access to the following resources:

- **Lists**: Manage contact lists and list memberships
- **Objects**: Access and manage CRM object definitions
- **Records**: Create, read, update, and delete CRM records
