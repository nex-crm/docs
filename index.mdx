## What is Nex

Nex is a context layer for AI agents. It unifies customer-facing signal from the tools where work actually happens — WhatsApp, email, Slack, meeting transcripts, support tickets, product events — into a single knowledge graph per workspace. AI agents use that graph as ground truth to answer natural-language queries, follow playbooks, and automate workflows.

The REST API lets you ingest context, query it, and wire your own agents into the same substrate Nex itself runs on.

## Quick Start

```bash

  -H "Authorization: Bearer sk-YOUR_API_KEY"
```
